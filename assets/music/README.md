# 音乐文件目录

节奏游戏需要以下 MP3 文件才能播放外部音乐：

| 文件名 | 曲目 | 说明 | BPM |
|--------|------|------|-----|
| `song1.mp3` | 长安少年行 | 李白主题曲 | 95 |
| `song2.mp3` | 凤求凰 | 李白传说皮肤 | 120 |
| `song3.mp3` | 永远的长安城 | 王者荣耀经典BGM | 140 |

## 如何添加音乐

1. 准备上述 MP3 文件（任意MP3均可，节拍谱面为预生成占位）
2. 将文件放入此目录 (`assets/music/`)
3. 刷新游戏页面，即可自动加载

## 注意事项

- 文件名必须与上表完全一致（`song1.mp3`、`song2.mp3`、`song3.mp3`）
- 支持任意 MP3 文件，当前节拍谱面基于 BPM 自动生成
- 如果缺少 MP3 文件，游戏会提供合成音乐作为备用方案
- 如需精确匹配音乐节拍，可在代码中编辑各曲目的 `notes` 数组
